Déploiement Heroku

Avant la séance (2h)

Vous devez commencer par créer votre compte sur Heroku.

Vous lirez la doc sur le déploiement d'application Spring Boot ici. Il faut télécharger le Heroku CLI.

Garder en tête que vous déploierez le projet serveur du cours. Vous n'avez pas à créer le projet Spring Boot. Ignorez donc la section "Creating a Spring Boot app"

Nous avons également fourni le Procfile pour le projet.

Déploiement

On parlera également des impacts des différents types d'instance

  • instance gratuite qui s'arrête et redémarre et perd ses fichiers
  • instance toujours en fonctionnement
  • multiples instances frontales avec une BD partagée

Nous discuterons de l'impact sur la BD mais également sur les sessions qui sont conservées en mémoire vive.
